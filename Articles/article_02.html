What Is Useful 

Maya’s built-in Rebuild Curve Options (found under Curves > Rebuild) allow you to adjust the number of spans and the degree of a curve, which works well for simple curve rebuilding. However, it does not provide control over individual span lengths, nor does it automatically adjust parameters based on each curve’s actual length. This limitation can make handling complex or varied curves more challenging.

In contrast, the Reset Curves tool offers greater flexibility and power. It enables you to select multiple curves at once and automatically calculates rebuild parameters based on the length of each curve. This means you can rebuild or create curves with specified distances between spans, giving you fine control over segment lengths. Additionally, you can generate curves where the distance between control vertices (CVs) is consistent across the entire curve, ensuring a more uniform and coherent curve structure.

The tool also includes an option to display Edit Points (EPs) and Control Vertices (CVs) on the curves, allowing you to visually inspect and edit curve details with ease, enhancing your workflow efficiency.

Designed to make curve rebuilding and recreation more efficient, precise, and customizable, this tool is especially useful when working with complex scenes containing multiple or intricate curves. It simplifies the process, improves the quality and consistency of results, and provides flexibility to meet diverse design and animation needs.

It’s also worth noting that differences in the number and distribution of CVs on curves can directly affect dynamics simulations or animation outcomes. Therefore, managing CV count and spacing carefully is crucial for achieving the desired final effect.



Reset Curves Options 



Reset Method 

There are two options for the Reset Method.


 
With the 'Rebuild Curve(s)' option, rebuild the curve using uniform parameterisation, allowing you to adjust the number of spans, the number of control vertices (CVs), and the degree of the curve(s). The rebuilt curve will closely follow the original shape by preserving the curvature.

With the 'Recreate Curve(s)' option, create a new curve where the distance between each CV point is equal, based on the original curve’s length. You can also change the number of spans, CVs, and degrees. Note that the resulting curve may not closely follow the original curvature, prioritising equal spacing between CVs instead.

Tip: If your goal is to create a straight curve or one with evenly spaced CVs regardless of the original shape, choose the Recreate Curve option.
 

Specify 

Use the Specify setting to define how curve segments are controlled during curve rebuilding or recreation.

Select Segment Length to specify the segment length when rebuilding or recreating curves. Select CV Amount to set the number of control vertices on the curve(s) when rebuilding or recreating curves.

Segment Length: Choose this option if you want to control the curve's rebuild based on a fixed segment length. The system will insert points along the curve at regular intervals according to the length you specify, ensuring consistent spacing between them. This is useful when uniform spacing is more important than the number of control points.

CV Amount: Choose this option if you prefer to specify the exact number of control vertices (CVs) for the rebuilt or recreated curve. This is useful when you need precise control over curve complexity or want to match the number of CVs across multiple curves.
 
     

Note: Use Segment Length for evenly spaced detail along curves (ideal for simulations or procedural generation). Use CV Amount when matching topology or achieving uniform editability across curves. If only one curve is selected, the Value on Shortest Curve will be applied directly to that curve.


Link / Unlink Value 

Toggle between Link and Unlink modes for the shortest and longest curve values.
Click to switch to , and when you change one value, the other value will automatically update. Switch to , you can enter the number you want individually.  



Note: Useful when you want either synchronised updates across curve values or full manual control over each one.


Value On Shortest Curve / Value On Longest Curve 

Allows you to manually specify the number of segments or CV points for the shortest and longest curves in your selection. For all intermediate-length curves, the corresponding values will be automatically interpolated based on their relative lengths.


P2: Segment Length On Shortest Curve = 0.2, Segment Length On Longest Curve = 1.5
P3: Segment Length On Shortest Curve = 0.4, Segment Length On Longest Curve = 0.4
P4: Segment Length On Shortest Curve = 0.4, Segment Length On Longest Curve = 0.2

Note: Ideal for maintaining proportional detail across multiple curves of varying lengths—ensuring smoother transitions and consistent topology distribution.


Display 

Displays the control vertices (CVs), edit points, or other components of the selected curve(s) in the viewport. This allows you to visualise and interact with the curve’s underlying structure for editing or inspection purposes.






Note: Useful when you need to fine-tune the shape of a curve or analyse its construction by making its editable components visible.


Degree 

The higher the Degree, the smoother the curve. The default setting (3 Cubic) works well for most curves.

 

Keep original curve(s) in a group 

When enabled, this option preserves the original curve(s) by placing them into a dedicated group. A new curve is then created based on the rebuilt version. This allows you to retain previous versions of the curve for easy comparison, especially when experimenting with different rebuild settings or options.



Note: Ideal for workflows where you want to try multiple rebuild configurations and compare results side-by-side without losing the original shape.
